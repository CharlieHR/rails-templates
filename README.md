# Rails Templates

Quickly generate a rails app with:
 - Devise (with a team_member model)
 - A TeamMember model
 - A Company model
 - Simple_form_for
 - Bootstrap 5
 - Stimulus


## How to use

```bash
rails new \
  --database postgresql \
  -m https://raw.githubusercontent.com/barangerbenjamin/rails-templates/master/template.rb \
  CHANGE_THIS_TO_YOUR_RAILS_APP_NAME
```
